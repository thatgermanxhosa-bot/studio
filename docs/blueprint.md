# **App Name**: StudioFlow

## Core Features:

- Homepage Video: Feature a captivating video background on the homepage to immediately grab visitors' attention and showcase the studio's cinematic style.
- Services Showcase: Display Pichulik Studio's service offerings: Videography, Photography, Post-Production and Styling.
- Work Portfolio: Dynamic, filterable gallery grid that showcases the diverse projects that the studio has worked on.
- Client Testimonials: A rotating carousel that highlights positive feedback that will build trust and credibility.
- Booking Form: Integrated booking system with calendar and scheduler for seamless client booking experiences.
- Quote Request AI: When processing a quote request, leverage AI to offer intelligently-customized packages. Use the contents of the project details, to intelligently estimate level of effort required. Provide a cost summary to studio staff. Provide this information to the studio as a tool for generating and refining their eventual quote.
- Contact Form: User-friendly contact form integrated with backend notification system to ensure prompt responses.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a sophisticated, artistic, and trustworthy feel.
- Background color: Dark navy (#120A26), almost black but with a hint of indigo, provides contrast to other elements
- Accent color: Soft lavender (#E6E6FA), as an analogous highlight that keeps the theme of quality in mind
- Body and headline font: 'Alegreya', a serif font lending an elegant, intellectual, and contemporary feel.
- Minimalist icons using the Lucide icon library for a consistent and clean aesthetic.
- Full-screen video on the homepage with an overlay, grid-based layouts for services and portfolio, and a clear, structured approach for contact information and forms.
- Smooth transitions and subtle animations for hover effects and content reveals to provide an engaging user experience.
